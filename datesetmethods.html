JavaScript setFullYear()

The setFullYear() method can optionally set month and day.

JavaScript setMonth()

The setMonth() method sets the month of a date object.

JavaScript setDate()

The setDate() method sets the day of a date object.

The setDate() method can also be used to add days to a date.

JavaScript setHours()

The setHours() method sets the hours of a date object.

JavaScript setMinutes()

The setMinutes() method sets the hours of a date object.
